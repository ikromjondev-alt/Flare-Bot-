# Discord Модератор-Бот

Бот на **discord.py 2.x** с базой данных **SQLite** — данные (мьюты, варны, настройки) не теряются при перезапуске.
Плюс отдельная **веб-панель управления** (Flask) и **мультиязычность** (RU/EN).

## Возможности

- Приветственное сообщение новому участнику (в стиле как на скриншоте) + счётчик "N-й участник"
- Авто-выдача роли новым участникам (мастер настройки в 4 шага)
- Модерация: `/mute`, `/unmute`, `/warn`, `/unwarn`, `/ban`, `/unban`, `/kick`
- `/report` — жалоба через выпадающее меню категорий + модальное окно + кнопки для приложения
  до 10 фото-доказательств
- `/support` — контакт поддержки
- `/help` — список всех команд
- `/language` — переключение языка бота (русский/английский) отдельно для каждого сервера
- Логи (детальные, в стиле карточек с полями и ID): вход/выход с сервера, вход/выход/переход
  между голосовыми каналами, серверный мьют/деафен, удаление и редактирование сообщений,
  создание/изменение (с цветом до/после)/удаление ролей, создание/изменение/удаление каналов
- Автоматическое удаление спам-ссылок (с настраиваемым списком разрешённых доменов)
- `/webapp` — приватная команда (видна только владельцу, ID указывается в `.env`) с инструкцией,
  как запустить и открыть веб-панель
- **Веб-панель** (`webpanel/`) — управление настройками через браузер: каналы, роли, язык,
  антиспам, авто-роль, список предупреждений и активных мьютов с возможностью снятия

## Установка бота

1. Создайте бота на https://discord.com/developers/applications, во вкладке **Bot**
   включите **Privileged Gateway Intents**: `SERVER MEMBERS INTENT` и `MESSAGE CONTENT INTENT`.
2. Пригласите бота на сервер со следующими правами (рекомендуется просто `Administrator`
   на старте, позже можно ограничить):
   `Manage Roles, Kick Members, Ban Members, Moderate Members, Manage Messages, View Channels, Send Messages, Embed Links, Attach Files, Read Message History, Connect`
3. Установите Python 3.10+ и зависимости:
   ```
   pip install -r requirements.txt
   ```
4. Скопируйте `.env.example` в `.env` и заполните `DISCORD_TOKEN` (и по желанию `GUILD_ID`
   для мгновенной синхронизации команд на одном сервере). Также заполните `OWNER_ID` своим
   Discord ID, если хотите пользоваться командой `/webapp`.
5. Запустите бота:
   ```
   python bot.py
   ```

## Настройка после запуска (на сервере, от имени администратора)

1. `/autorole setup` → `/autorole channel #канал` → `/autorole role @роль` →
   (необязательно) `/autorole message текст` → `/autorole enable`
2. `/logsettings channel #канал-логов`
3. `/reportsettings channel #канал-жалоб`
4. `/antispam on` (включён по умолчанию) и при необходимости `/antispam allow домен.com`
5. `/language` — выбрать русский или английский язык для сервера

## Веб-панель управления

Панель — отдельное Flask-приложение, читающее ту же базу `bot_database.db`.
Запускать нужно на той же машине (или с доступом к тому же файлу БД), что и бот.

1. В `.env` заполните `WEBPANEL_PASSWORD` и `WEBPANEL_SECRET_KEY`.
2. Запустите:
   ```
   cd webpanel
   python app.py
   ```
3. Откройте `http://localhost:5000`, войдите по паролю.
4. На дашборде появятся сервера, на которых бот уже что-то настраивал (роли, каналы и т.д.).
   Там же можно: менять каналы жалоб/логов, роль Muted, язык бота, включать/выключать антиспам,
   настраивать авто-роль и приветствие, управлять списком разрешённых доменов, снимать
   предупреждения и мьюты.

**Важно:** если вы снимаете мьют через веб-панель, запись удаляется из базы, но саму роль
"Muted" в Discord снимет только сам бот (при следующей проверке/перезапуске). Для мгновенного
снятия используйте команду `/unmute` в Discord.

## Структура проекта

```
bot.py                    — точка входа бота
config.py                 — загрузка настроек из .env
database.py                — работа с SQLite (все данные бота, асинхронно)
i18n.py                     — словари переводов RU/EN
cogs/
  welcome.py                 — приветствие новых участников
  autorole.py                — мастер настройки авто-роли
  moderation.py               — mute/unmute/warn/unwarn/ban/unban/kick
  report.py                    — /report (меню категорий + модалка + кнопки для фото)
  support.py                    — /support
  help_cmd.py                    — /help
  logging_cog.py                  — все логи сервера
  antispam.py                      — фильтр спам-ссылок
  language.py                       — /language
webpanel/
  app.py                              — Flask-приложение веб-панели
  db.py                                — синхронный доступ к той же SQLite базе
  templates/                           — HTML-шаблоны панели
```

## Примечания

- Для тайм-мьютов используйте формат `3d` (дни) или `30m` (минуты), например:
  `/mute @user 3d Спам в чате`. Оставьте поле времени пустым — мьют будет перманентным.
- Роль "Muted" создаётся автоматически при первом использовании `/mute`, если её ещё нет.
- Все настройки, мьюты и варны сохраняются в `bot_database.db` и переживают перезапуск бота.
- Веб-панель и бот — два независимых процесса; запускайте их отдельно (`python bot.py` и
  `python webpanel/app.py`), можно на одном сервере через `pm2`, `systemd` или `screen/tmux`.
